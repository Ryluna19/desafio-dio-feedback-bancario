# Prompt Final — Extraindo Insights de Feedbacks Bancários

Atue como um analista de experiência do cliente em uma instituição bancária.

Quero que você analise feedbacks de clientes sobre serviços bancários, como aplicativo, Pix, cartão de crédito, conta corrente, atendimento digital e suporte ao cliente. O objetivo é identificar os principais temas comentados, reclamações frequentes, elogios, sentimento dos clientes e oportunidades de melhoria.

Essa análise será usada por equipes de experiência do cliente, atendimento e produtos digitais para entender melhor os problemas relatados pelos clientes e apoiar decisões de melhoria nos canais e serviços do banco.

Os dados fornecidos podem conter informações como data do comentário, canal de atendimento, produto ou serviço citado, texto do feedback e nota de satisfação, quando disponível.

Ao fazer a análise, siga estas orientações:

1. Classifique os feedbacks por tema principal, como aplicativo, Pix, cartão, tarifas, segurança, conta, atendimento ou suporte.
2. Identifique o sentimento de cada comentário como positivo, neutro ou negativo.
3. Classifique a urgência como baixa, média ou alta, considerando o impacto que o problema pode ter para o cliente.
4. Destaque padrões recorrentes, principais reclamações, elogios e possíveis oportunidades de melhoria.
5. Dê atenção aos comentários negativos, principalmente quando envolverem falhas de segurança, problemas em transações, dificuldade de acesso ou atendimento ruim.
6. Use exemplos curtos dos próprios comentários como evidência, sem expor informações pessoais.
7. Sugira ações práticas que possam ajudar as equipes responsáveis a melhorar a experiência do cliente.

Organize a resposta no seguinte formato:

* Um resumo executivo com até 5 linhas.
* Uma tabela com: tema, sentimento predominante, urgência, evidência encontrada e ação sugerida.
* Uma lista com os principais problemas identificados.
* Uma lista com os principais elogios identificados.
* Uma lista com oportunidades de melhoria.
* Uma lista final com as 3 prioridades mais importantes.

Restrições:

* Use apenas os dados fornecidos.
* Não invente números, causas, percentuais ou conclusões sem evidência.
* Não exponha dados pessoais ou sensíveis, como nome completo, CPF, telefone, e-mail, endereço, número de conta ou cartão.
* Caso encontre dados sensíveis nos comentários, substitua por “[dado removido]”.
* Se os dados não forem suficientes para alguma conclusão, informe essa limitação.
* Use uma linguagem simples, clara e voltada para tomada de decisão.
